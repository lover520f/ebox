## 🚀 使用步骤

### 1. 下载便携版

下载 `ebox-v1.0.0-portable.zip`

### 2. 解压文件

右键 → 全部解压缩到任意目录

### 3. 运行程序

双击 `ebox_client.exe` 启动

### 4. 连接 Emby

1. 点击"开始使用"
2. 添加服务器：`http://IP 地址：8096`
3. 保存并连接
4. 开始观影！
