# E 宝盒 v2.0.0 需求规格说明书

## 项目概述

E 宝盒是一个跨平台 Emby 媒体服务器客户端，参考 CapyPlayer 和 Emby 官方项目设计，支持 Windows、Android TV、iOS 平台。

## 参考项目

1. **CapyPlayer** (https://capyplayer.feifeiduck.com/zh)
   - 支持 Emby/Plex/Jellyfin
   - 4K HDR、杜比视界、杜比全景声
   - 专业级解码引擎
   - 简洁直观的 UI

2. **Emby Official** (https://github.com/MediaBrowser)
   - Emby.Theater - 官方桌面客户端
   - Emby.Android - 官方 Android 客户端
   - jellyfin-web - Web 客户端

## 非功能需求

### 性能
- 冷启动时间 < 2 秒
- 页面切换 < 300ms
- 视频加载 < 3 秒 (局域网)
- 内存占用 < 200MB

### 兼容性
- Windows 10/11
- Android TV 8.0+
- iOS 14+

### 安全
- HTTPS 支持
- Token 加密存储
- 不记录观看历史 (隐私模式)
