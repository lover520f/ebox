# E 宝盒 (ebox) - 第一阶段 MVP 需求文档

## 1. 产品概述

E 宝盒是一款跨平台媒体中心应用，支持连接 Emby 服务器和本地媒体库，提供视频、音频、电子书、漫画等多种媒体类型的播放和管理功能。

## 2. 第一阶段 (MVP) 范围

### 2.1 支持平台
- ✅ Windows 10/11 桌面端
- ⏸️ Android 手机端 (第二阶段)
- ⏸️ iOS 端 (第二阶段)
- ⏸️ Android TV 端 (第三阶段)

### 2.2 核心功能

#### 2.2.1 Emby 服务器连接
- 添加/编辑/删除 Emby 服务器配置
- 支持 HTTP/HTTPS 连接
- 支持用户名/密码认证
- 支持 API Key 认证
- 自动获取媒体库列表
- 用户认证和会话管理

#### 2.2.2 视频播放
- Emby 服务器视频流播放
- 本地视频文件播放
- 支持格式：MP4, MKV, AVI, WMV, FLV
- 播放控制：播放/暂停/快进/快退
- 进度记忆（本地存储）
- 字幕支持：SRT, ASS（外挂字幕）
- 倍速播放：0.5x - 2.0x
- 画质选择（Emby 服务器转码）

#### 2.2.3 媒体库浏览
- 海报墙展示（网格/列表视图切换）
- 媒体详情页面
  - 标题、简介、演职员表
  - 评分、年份、类型标签
  - 相关推荐
- 分类筛选：电影、电视剧、综艺、动漫
- 搜索功能：按标题搜索

#### 2.2.4 本地媒体管理
- 扫描指定文件夹
- 自动识别视频文件
- 基础元数据获取（文件名解析）
- 最近播放记录

### 2.3 UI/UX 要求

#### 设计语言
- 灵感来源：Hills 播放器
- 深色主题优先（默认）
- 渐变色点缀（紫色/蓝色）
- 圆角卡片式设计
- 平滑过渡动画

#### 核心页面
1. **欢迎页** - 首次启动引导
2. **主页** - 媒体库入口
3. **服务器管理页** - Emby 服务器配置
4. **媒体库页** - 海报墙展示
5. **媒体详情页** - 详情和播放按钮
6. **播放器页** - 全屏播放界面
7. **本地媒体页** - 本地文件浏览
8. **设置页** - 应用设置

### 2.4 非功能需求

#### 性能
- 冷启动时间 < 3 秒
- 页面切换流畅（60fps）
- 视频加载时间 < 2 秒（本地网络）
- 内存占用 < 300MB

#### 兼容性
- Windows 10 64 位及以上
- 屏幕分辨率：1920x1080 及以上
- 支持 4K 显示输出

## 3. 用户故事

### 3.1 首次使用
- **作为** 新用户
- **我想** 在启动时看到简洁的设置向导
- **以便** 快速配置 Emby 服务器或选择本地播放

### 3.2 连接 Emby 服务器
- **作为** Emby 用户
- **我想** 添加服务器地址和账号
- **以便** 访问我的媒体库

### 3.3 浏览电影
- **作为** 观影用户
- **我想** 看到精美的海报墙
- **以便** 快速找到想看的电影

### 3.4 播放视频
- **作为** 观众
- **我想** 点击海报即可播放
- **以便** 获得流畅的观影体验

### 3.5 播放进度记忆
- **作为** 连续剧观众
- **我想** 下次打开时自动定位上次观看位置
- **以便** 继续观看

### 3.6 本地播放
- **作为** 本地媒体用户
- **我想** 浏览和播放本地视频文件
- **以便** 无需服务器也能观影

## 4. 验收标准

### 4.1 功能验收
- [ ] 成功连接 Emby 服务器并获取媒体库
- [ ] 海报墙正确展示媒体内容
- [ ] 视频播放流畅，无明显卡顿
- [ ] 播放进度正确保存和恢复
- [ ] 本地视频扫描和播放正常
- [ ] 字幕加载和切换正常
- [ ] 搜索功能返回正确结果

### 4.2 性能验收
- [ ] 启动时间 < 3 秒
- [ ] 页面切换 60fps
- [ ] 内存占用 < 300MB
- [ ] 视频加载 < 2 秒

### 4.3 UI 验收
- [ ] 深色主题默认启用
- [ ] 渐变色和圆角设计统一
- [ ] 动画过渡流畅
- [ ] 响应式布局适配不同窗口尺寸

## 6. 术语表

| 术语 | 说明 |
|------|------|
| Emby | 开源媒体服务器软件 |
| 海报墙 | 以网格形式展示媒体封面的 UI |
| 转码 | 服务器将视频转换为不同格式/画质 |
| 外挂字幕 | 独立的字幕文件（.srt/.ass） |
