# 🎯 点击这里开始构建！

## 快速步骤（3 步完成）

### 第 1 步：访问 Actions 页面

点击这个链接：
```
https://github.com/lover520f/ebox/actions
```

### 第 2 步：选择工作流

在页面左侧点击：
- **"Build Windows Release"** 或 **"Windows Build and Release"**

### 第 3 步：运行工作流

1. 点击 **"Run workflow"** 按钮（绿色）
2. 确认版本号：`v1.0.0`
3. 点击 **"Run workflow"** 按钮

---

## ⏱️ 等待完成

- **预计时间**: 10-15 分钟
- **实时日志**: 可以点击正在运行的任务查看
- **完成后**: 自动跳转到 Release 页面

---

## 📥 下载

完成后访问：
```
https://github.com/lover520f/ebox/releases/tag/v1.0.0
```

下载 `ebox-v1.0.0-portable.zip`
