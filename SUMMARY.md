## 💡 技术亮点

### 1. Hills 风格设计
- 深蓝黑色背景 (#0F172A)
- 紫蓝色渐变点缀
- 大圆角卡片设计
- 清晰的字体层次

### 2. 现代化架构
- **MVVM 模式**: 清晰的 MVC 分层
- **Provider 状态管理**: 响应式数据流
- **Dio HTTP 客户端**: 高性能网络请求
- **Hive 本地存储**: 轻量级 NoSQL 数据库

### 3. Emby API 集成
- 完整的 API 封装
- 认证和会话管理
- 媒体库扫描
- 播放进度同步
- 图片 CDN 加速

### 4. 跨平台设计
- 一套代码多平台运行
- 响应式布局
- 屏幕自适应
- 桌面端和移动端优化
