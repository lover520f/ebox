## ❓ 为什么需要手动启用？

GitHub 出于安全考虑，对新创建仓库的 Actions 功能有以下限制：

1. 新仓库默认可能禁用 Actions
2. 防止恶意工作流被自动执行
3. 需要仓库所有者明确授权

这是 GitHub 的正常安全机制。
