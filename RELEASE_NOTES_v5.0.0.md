## 🔧 技术规格

### 系统要求

**Windows 版**
- Windows 10 或更高版本
- 4GB RAM 以上
- 支持 DirectX 11 的显卡

**Android Mobile**
- Android 8.0 (API 26) 或更高
- 2GB RAM 以上
- 支持 ARM64 或 ARMv7

**Android TV**
- Android TV 8.0 (API 26) 或更高
- Google TV 支持
- 遥控器支持

### Emby 服务器要求
- Emby Server 4.7+
- 网络可达
- 用户账号

## 📱 安装说明

### Windows 版
1. 下载 `ebox-windows-v5.0.0.zip`
2. 解压到任意目录
3. 运行 `E 宝盒.exe`
4. 添加 Emby 服务器地址

### Android Mobile 版
1. 下载 `ebox-mobile-v5.0.0.apk`
2. 允许安装未知来源应用
3. 安装 APK
4. 打开应用添加服务器

### Android TV 版
1. 下载 `ebox-tv-v5.0.0.apk`
2. 通过 USB 或文件管理器安装
3. 或使用 Sideload Launcher
4. 打开应用添加服务器

## 🎮 使用教程

### 快速开始
1. 首次启动显示欢迎页面
2. 点击"添加服务器"
3. 输入 Emby 服务器 URL
4. 输入用户名和密码（可选）
5. 测试连接后保存

### 浏览媒体
- **首页**: 继续观看 + 最近添加 + 推荐
- **电影**: 所有电影库
- **剧集**: 所有剧集库
- **本地**: 本地媒体（开发中）

### 播放控制

**Windows 键盘:**
- `空格` - 播放/暂停
- `←/→` - 后退/前进 10 秒
- `↑/↓` - 音量增减
- `F` - 全屏切换
- `M/K` - 静音
- `ESC` - 返回

**Mobile 触控:**
- 左侧上下滑动 - 亮度
- 右侧上下滑动 - 音量
- 横向拖动 - 快进/退
- 双击 - 播放/暂停
- 单击 - 显示控制栏

**TV 遥控器:**
- 方向键 - 导航
- OK 键 - 选中
- 播放/暂停键 - 播放控制
- 返回键 - 返回上一级
