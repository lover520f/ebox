## 📥 安装说明

### 方法一：便携版 (推荐)

1. 下载 `ebox-v1.0.0-windows.zip`
2. 解压到任意目录（如 `D:\ebox`）
3. 双击运行 `ebox_client.exe`
4. 开始使用！

**优点**: 
- 无需安装
- 不写注册表
- 可放 U 盘随身携带

### 方法二：安装版 (待发布)

1. 下载 `ebox-v1.0.0-setup.exe`
2. 双击运行安装程序
3. 选择安装目录
4. 完成安装

## 🚀 快速开始

### 1. 连接 Emby 服务器

```
1. 启动 E 宝盒
2. 点击"开始使用"
3. 点击右上角"+"添加服务器
4. 填写：
   - 服务器地址：http://你的 IP:8096
   - 服务器名称：我家里的 Emby（可选）
   - 用户名/密码（如果有）
5. 点击"测试连接"
6. 保存并连接
```

### 2. 播放电影

```
1. 在主页点击"电影"库
2. 浏览海报墙
3. 点击电影海报查看详情
4. 点击"播放"按钮
5. 享受观影时光！
```

### 3. 播放本地视频

```
1. 底部导航点击"本地"
2. 点击"选择文件夹"
3. 选择包含视频的目录
4. 自动扫描视频文件
5. 点击任意视频播放
```

## 🎮 使用说明

### 播放控制

| 操作 | 功能 |
|------|------|
| 单击屏幕任意位置 | 显示/隐藏控制栏 |
| 双击屏幕左侧 | 快退 10 秒 |
| 双击屏幕右侧 | 快进 10 秒 |
| 点击 ▶/⏸ | 播放/暂停 |
| 拖动进度条 | 跳转播放位置 |
| 点击 1.0x | 调整播放速度 |
| Esc 键 | 退出全屏 |

### 快捷键

| 按键 | 功能 |
|------|------|
| Space | 播放/暂停 |
| ← | 快退 10 秒 |
| → | 快进 10 秒 |
| ↑ | 音量 + |
| ↓ | 音量 - |
| F | 全屏切换 |
| Esc | 退出 |
