### 🚀 如何运行项目

```bash
# 1. 克隆项目
git clone https://github.com/lover520f/ebox.git
cd ebox/client

# 2. 安装依赖
flutter pub get

# 3. 运行 (Windows)
flutter run -d windows
```

### 📦 如何打包

```bash
# Windows 打包
flutter build windows --release

# 输出目录
build/windows/runner/Release/
```

## 📱 使用方法

1. **连接 Emby 服务器**
   - 添加服务器
   - 填入地址 `http://IP:8096`
   - 测试连接
   - 保存

2. **看电影**
   - 选择媒体库
   - 点击电影海报
   - 点击播放

3. **播放本地视频**
   - 底部导航点"本地"
   - 选择视频文件夹
   - 点击视频播放
